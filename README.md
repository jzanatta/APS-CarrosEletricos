🌐 **[Visualizar o site ao vivo](https://aps-carros-eletricos.netlify.app/)**

---

## 📋 Sobre o Projeto

Site desenvolvido como entrega da disciplina de **Atividades Práticas Supervisionadas (APS)** da UNIP, com o tema **"Desenvolvimento de um Site em XHTML"** voltado ao desenvolvimento sustentável.

O projeto aborda os carros elétricos como solução para a redução de emissões de gases poluentes, apresentando de forma clara e interativa o funcionamento da tecnologia, suas vantagens, desafios e impacto ambiental.

## 🛠️ Tecnologias Utilizadas

- **XHTML 1.0 Transitional** — estrutura e marcação semântica
- **CSS 2.1 / CSS3** — estilos, animações e layout responsivo
- **JavaScript ES6+** — interatividade, scroll reveal e animações
- **Google Fonts** — tipografias Orbitron e Sora
- **Netlify** — hospedagem e deploy contínuo

---

## ✨ Funcionalidades

- **Modo claro / escuro** — toggle estilo switch com animação suave; tema escuro como padrão
- **Scroll Reveal** — elementos animados conforme o usuário desce a página
- **Navbar inteligente** — se esconde ao rolar para baixo e reaparece ao subir
- **Contadores animados** — estatísticas no hero com incremento progressivo
- **Seção de desafios interativa** — clique nos itens para trocar a imagem lateral
- **Barras de progresso animadas** — métricas visuais na conclusão
- **Design responsivo** — adaptado para desktop, tablet e mobile
- **Menu hamburguer** — navegação mobile funcional

## 📖 Como Executar Localmente

Não há dependências ou processo de build. Basta clonar e abrir o arquivo:

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
cd seu-repositorio
# Abra o index.html no navegador
```

> O site usa apenas arquivos estáticos (HTML + CSS + JS inline), sem necessidade de servidor.
